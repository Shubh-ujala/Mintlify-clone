# Mintlify Clone

A static page inspired by Mintlify. Built with plain HTML and CSS.

## Preview
Open `index.html` in your browser to view the page.

## Features
- Hero section with CTA input
- Sponsor logo grid
- Feature cards and assistant highlight
- Enterprise knowledge section with banner

## Tech Stack
- HTML5
- CSS3

## Project Structure
- `index.html`
- `style.css`
- `assets/`

## Notes
- Fonts are loaded from Google Fonts in the HTML head.
- Images and SVGs are stored in the `assets` folder.
